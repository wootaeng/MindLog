# 개인정보 처리방침 (Privacy Policy)

**시행일:** 2026년 4월 1일

## 1. 개인정보의 처리 목적
본 애플리케이션("MindLog" 또는 "앱")은 사용자의 편의를 제공하기 위해 다음과 같은 목적으로 최소한의 정보를 접근 및 처리합니다.
- 기기 로컬 저장소: 사용자의 일기 데이터를 기기 내부에 저장하기 위해 사용됩니다.
- 오디오 권한: 음성 인식을 통한 일기 작성 기능을 위해 사용됩니다.

앱은 사용자의 개인정보(일기 내용 등)를 외부 서버로 수집, 저장, 전송하지 않습니다. 모든 데이터는 사용자의 기기(로컬)에만 저장됩니다.

## 2. 외부 서비스 연동
앱 내 광고 제공 및 AI 기능을 위해 외부 서비스가 사용될 수 있으며, 해당 서비스의 정책을 따릅니다.
- **Google AdMob:** 광고 제공을 위해 기기 식별자 및 광고 데이터를 수집할 수 있습니다. (구글 개인정보 처리방침 참고)
- **Google Gemini API:** 사용자가 입력한 텍스트 데이터 형태를 AI 분석 목적으로 전송할 수 있습니다. 해당 데이터의 취급은 Google의 정책을 따릅니다.

## 3. 권한 안내
원활한 앱 사용을 위해 다음과 같은 권한을 요청할 수 있습니다.
- `RECORD_AUDIO`: 음성 입력을 통한 텍스트 변환 시 사용
- `POST_NOTIFICATIONS`: 설정한 시간의 푸시 알림 제공 시 사용
- `SCHEDULE_EXACT_ALARM`: 정확한 시간에 알림을 보내기 위해 사용

## 4. 데이터 파기
사용자가 앱을 삭제하는 경우, 기기에 로컬로 저장된 모든 애플리케이션 데이터(일기 포함)는 즉시 파기됩니다.
